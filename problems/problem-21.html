Problem 21

Consider the function $f:(-\infty,\infty) \to (-\infty,\infty)$ defined by $f(x) = \frac{x^2-ax+1}{x^2+ax+1} ,0\lt a \lt 2$
Which of the following is true ?
a)$(2+a)^2f''(1)+(2-a)^2f''(-1)=0$
b)$(2-a)^2f''(1)-(2+a)^2f''(-1)=0$
c) $f'(1)f'(-1)=(2-a)^2$
d) $f'(1)f'(-1)=-(2+a)^2$

Solution

$$f'(x)=\frac{(x^2+ax+1)(2x-a)-(x^2-ax+1)(2x+a)}{(x^2+ax+1)^2}$$ $$=\frac{(\cancel{2x^3}+2ax^2+\cancel{2x}-ax^2-\cancel{a^2x}-a)-(\cancel{2x^3}-2ax^2+\cancel{2x}+ax^2-\cancel{a^2x}+a)}{(x^2+ax+1)^2}$$ $$=\frac{2a(x^2-1)}{(x^2+ax+1)^2}$$ $$f''(x)=\frac{(x^2+ax+1)^2(4ax)-(2ax^2-2a)2(x^2+ax+1)(2x+a)}{(x^2+ax+1)^4}$$ $$f''(1)=\frac{4a}{(2+a)^2}$$ $$f''(-1)=-\frac{4a}{(2-a)^2}$$

Option a) $(2+a)^2f''(1)+(2-a)^2f''(-1)=0$
$$\cancel{(2+a)^2}\frac{4a}{\cancel{(2+a)^2}}+\cancel{(2-a)^2}\frac{-4a}{\cancel{(2-a)^2}}=4a-4a=0$$ $$f'(1)=f'(-1)=0$$
a) is the Correct Option.
