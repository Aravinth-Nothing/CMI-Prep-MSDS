Problem 27

If $\log_{\sqrt{3}} x + \log_{\sqrt[4]{3}} x + \log_{\sqrt[6]{3}} x + ...\log_{\sqrt[16]{3}} x = 36$ then the value of $x$ is ?
a) $3$
b) $\sqrt{3}$
c) $4\sqrt{3}$
d) $9$

Solution

$$\frac{1}{\log_x{\sqrt{3}}}+\frac{1}{\log_x{\sqrt[4]{3}}}+\frac{1}{\log_x{\sqrt[6]{3}}}+...+\frac{1}{\log_x{\sqrt[16]{3}}}=36$$ $$\frac{2}{\log_x 3}+\frac{4}{\log_x 3}+\frac{6}{\log_x 3}+...+\frac{16}{\log_x 3}=\frac{2(1+2+3+...+8)}{\log_x 3}$$ $$=\frac{2(36)}{\log_x 3}=36$$ $$\log_x 3 =2$$ $$x^2=3$$ $$x=\sqrt{3}$$
