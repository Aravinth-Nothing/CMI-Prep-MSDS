Problem 19

Total number of local maxima and local minima of the function, $$f(x)= \left\{ \begin{array}{ll} (2x+3)^3 & ,-3 \lt x \leq -1 \\ x^{\frac{2}{3}} & ,-1 \lt x \lt 2 \end{array} \right. $$ is,
a) 0
b) 1
c) 2
d) 3

Solution

$$f'(x)=\left\{\begin{array}{ll} 6(2x+3)^2 & , -3 \lt x \leq -1 \\ \frac{2}{3}x^{-\frac{1}{3}} & ,-1 \lt x \lt \end{array}\right.$$ $$f'(x)\gt 0 \forall -3 \lt x \leq-1 $$

One Local Maxima is Here.
$$f'(x)\lt 0 \forall -1 \lt x \leq 0 $$
One Local Minima is Here.
$$f'(x)\gt 0 \forall 0 \lt x \leq 2 $$
$\therefore$ The total number of local maxima and local minima of the function is 2.
c) is the Correct Option.
