Problem 18

Let $f$ and $g$ be real valued functions defined on interval $(-1, 1)$ such that $g''(x)$ is continuous, $g(0)\neq0, g'(0) = 0, g''(0) \neq 0$ and $f(x) = g(x) \sin x$.
STATEMENT-1 : $\displaystyle \lim_{x\to 0 }[g(x)\cot x - g(0) \csc x]$
STATEMENT-2: $f'(0) = g(0)$

(a) STATEMENT-1 is True, STATEMENT-2 is True ; STATEMENT-2 is a correct explanation for STATEMENT-1
(b) STATEMENT-1 is True, STATEMENT-2 is True ; STATEMENT-2 is NOT a correct explanation for STATEMENT-1
(c) STATEMENT-1 is True, STATEMENT-2 is False
(d) STATEMENT-1 is False, STATEMENT-2 is True

Solution

$$f'(x)=g(x)\cos x +g'(x) \sin x$$ $$f'(0)=g(0).1+g'(0).0=g(0)$$ $$f'(0)=g(0)$$

STATEMENT-2 is True.
$$f''(x)=g'(x)\cos x-g(x)\sin x+g''(x)\sin x+g'(x)\cos x$$ $$f''(0)=g'(0)\cos 0-g(0)\sin x+g''(0)\sin 0+g'(0)\cos 0$$ $$f''(0)'=0$$ From STATEMENT-1: $$\displaystyle \lim_{x\to 0 }[g(x)\cot x - g(0) \csc x]= \lim_{x\to 0}\left[\frac{g(x)\cos x-g(0)}{\sin x}\right]$$
Using L'Hospital's Rule, Differentiate.
$$= \lim_{x\to 0}\left[\frac{g'(x)\cos x-g(x)\sin x}{\cos x}\right]=\left[\frac{g'(0)\cos 0-g(0)\sin 0}{\cos 0}\right]=0$$ $$\displaystyle \lim_{x\to 0 }[g(x)\cot x - g(0) \csc x]=f''(0)=0$$
STATEMENT-1 is True.

(b) STATEMENT-1 is True, STATEMENT-2 is True ;
STATEMENT-2 is NOT a correct explanation for STATEMENT-1
