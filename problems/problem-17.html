Problem 17

Let $f(x)$ be a non-constant twice differentiate function defined on $(-\infty,\infty)$ such that $f(x)=f(x-1)$ and $f'(\frac{1}{4})=0$. Then,
a) $f''(x)$ vanishes at least twice on $[0,1]$
b) $f'(\frac{1}{2})=0$
c) $\int_{-\frac{1}{2}}^{\frac{1}{2}}f\left(x+\frac{1}{2}\right)\sin x dx =0 $
d) $\int_{0}^{\frac{1}{2}}f(t)e^{\sin{\pi t}}dt=\int_{\frac{1}{2}}^{1}f(1-t)e^{\sin{\pi t}}dt$

Solution

Given $f(x)=f(1-x) $, substituting $x \to x+\frac{1}{2}$ gives, $f(\frac{1}{2}+x)=f(\frac{1}{2}-x)$.
We can see that, $f(x)$ is an even function symmetric about $x=0$.
$\therefore$ at $x=0$ , $f'(\frac{1}{2})=0$

b) is a correct option.

Given $f'(\frac{1}{4})=0$ , Applying $x=\frac{1}{4}$ in $f(x)$, we get, $f(\frac{1}{4})=f(\frac{3}{4}).$
$\therefore f'(\frac{3}{4})=0.$ We can see that $f'(x)=0$ for $x=\frac{1}{4},\frac{1}{2},\frac{3}{4}$
a) is a correct option.

We know that, $f(\frac{1}{2}+x)=f(\frac{1}{2}-x)$ is an even function.
$f(\frac{1}{2}+x)\sin x$ is an odd function.
Note: $\int_{-a}^{+a} f(x)dx = 0$ if $f(x)$ is an odd function.
$$\therefore \int_{-\frac{1}{2}}^{\frac{1}{2}}f\left(x+\frac{1}{2}\right)\sin x dx =0 $$
c) is a correct option.
Take $\int_{0}^{\frac{1}{2}}f(t)e^{\sin{\pi t}}dt$, subsitute $t \to (1-t)$
$$=\int_{1-0}^{1-\frac{1}{2}}f(1-t)e^{\sin{\pi (t-1)}}d(1-t)=-\int_{1}^{\frac{1}{2}}f(1-t)e^{\sin{ (\pi t-\pi)}}dt$$ $$=\int_{\frac{1}{2}}^{1}f(1-t)e^{\sin{ \pi t}}dt$$
d) is a correct option.

a),b),c),d) are all correct.
