Problem 23

Let $f:\left(-\frac{\pi}{2},\frac{\pi}{2}\right)\to\mathbb{R}$ be given by $$f(x)=(\log(\sec x+\tan x))^3$$ Then,
a) $f(x)$ is an odd function
b) $f(x)$ is an one-one function
c) $f(x)$ is an onto function
d) $f(x)$ is an even function

Solution

Checking Even or Odd Function,
$$f(-x)=(\log(\sec (-x)+\tan (-x)))^3=(\log(\sec x-\tan x))^3$$ Note: $\sec^2 x-\tan^2 x=(\sec x -\tan x)(\sec x+\tan x)=1$
$\sec x -\tan x = \frac{1}{\sec x +\tan x}$ $$=\left(\log\left(\frac{1}{\sec x +\tan x}\right)\right)^3=\left(\log\left(\sec x +\tan x\right)^{-1 }\right)^3=\left(-\log\left(\sec x +\tan x\right)\right)^3$$ $$=-\left(\log\left(\sec x +\tan x\right)\right)^3=-f(x)$$

$f(x)$ is an odd function.

$\log$, $x^3$ function is a strictly increasing function.
differentiating $(\sec x +\tan x)$,
$$\frac{d}{dx}(\sec x+\tan x)= \sec x(\sec x +\tan x)$$
in the domain, $\left(-\frac{\pi}{2},\frac{\pi}{2}\right)$ the function, $\sec x(\sec x +\tan x) \gt 0$
$\therefore f(x)$ is a strictly increasing function, hence an one-one Function.
$f(x)$ in range $\left(-\frac{\pi}{2},\frac{\pi}{2}\right)$ takes all values in $\mathbb{R}$, hence $f(x)$ is an onto function.
a),b),c) are the correct Options.
