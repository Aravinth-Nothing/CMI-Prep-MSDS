Problem 24

Let $f:\mathbb{R}\to\mathbb{R}$ and $g:\mathbb{R}\to\mathbb{R} $ be respectively given by,
$f(x)=|x|+1$
$g(x)=x^2+1$
We Define $h:\mathbb{R}\to\mathbb{R}$ by $h(x)=\left\{\begin{array}{ll} \max {{f(x),g(x)}} & , x \leq 0 \\ \min {f(x),g(x)} & , x \gt 0 \end{array}\right.$
In how many number of points $h(x)$ is not differentiable.

Solution

The Green curve Shown (In Light Mode) in the plot above is the plot of $h(x)$.
As we can see the slope of the curve changes drastically in 3 points. In those points the function $h(x)$ is not differentiable. $$\therefore$$ the function is not differentiable at 3 points.
Answer: 3
