Problem 11

If $\sin\theta_1-\sin\theta_2=a, \cos\theta_1+\cos\theta_2=b$ then,
a) $a^2+b^2\geq 4$
b) $a^2+b^2\leq 4$
c) $a^2+b^2\geq 3$
d) $a^2+b^2\leq 2$

Solution

Squaring given equations, $$a^2=\sin^2\theta_1+\sin^2\theta_2-2\sin\theta_1\sin\theta_2 \label{one}\tag{1}$$ $$b^2=\cos^2\theta_1+\cos^2\theta_2+2\cos\theta_1\cos\theta_2 \label{two}\tag{2}$$ $$a^2+b^2=(\sin^2\theta_1+\cos^2\theta_1)+(\sin^2\theta_2+\cos^2\theta_2)-2\sin\theta_1\sin\theta_2+2\cos\theta_1\cos\theta_2$$ Note: $\sin^2\theta+\cos^2\theta=1$ $$a^2+b^2=1+1+2(\cos\theta_1\cos\theta_2-\sin\theta_1\sin\theta_2)$$ $$a^2+b^2=2+2(\cos(\theta_1-\theta_2))$$ Minimum value of $\cos(\theta_1-\theta_2)=-1$:
Hence the Minimum value of $a^2+b^2 = 2+2(-1) =2-2 =0$
Maximum value of $\cos(\theta_1-\theta_2)=1$:
Hence the Maximum value of $a^2+b^2 = 2+2(1) =2+2 =4$

Hence b) $a^2+b^2\leq 4$ is the Correct answer.
